# MyGPSapplication

MyGPSapplication est une application Android simple qui permet de récupérer et d'afficher la position GPS de l'utilisateur.

## Fonctionnalités

- Récupération de la latitude et de la longitude
- Affichage des coordonnées GPS
- Envoi des coordonnées vers un second écran

## Technologies

- Java
- Android Studio
- Android SDK
- LocationManager

## Permissions

- ACCESS_FINE_LOCATION
- ACCESS_COARSE_LOCATION

## Lancer le projet

1. Cloner le dépôt
2. Ouvrir le projet dans Android Studio
3. Lancer l'application sur un émulateur ou un téléphone
4. Autoriser l'accès à la localisation
